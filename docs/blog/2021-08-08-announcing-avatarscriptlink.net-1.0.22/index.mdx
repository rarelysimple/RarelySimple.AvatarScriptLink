AvatarScriptLink.NET version 1.0.22 is [now available on NuGet](https://www.nuget.org/packages/RarelySimple.AvatarScriptLink/1.0.22).
This is a maintenance release which includes the following updates:

* Addresses security vulnerabilities from various project dependencies.
* Converting OptionObject content to HTML has been improved.
* Miscellaneous code quality improvements.
